Fast Automated Tests

Many teams recognize that because of how quickly unit tests can execute, unit tests should make up the majority of their test suite.1 However, when they attempt to write unit tests for their code, they may not know where to start or their code may in fact be untestable. I help teams identify parts of their code base that are not amenable to unit testing and suggest refactoring methods that will make the code more unit testable.

I can help your team identify repetitive tasks that are susceptible to automation. I have extensive experience configuring Jenkins CI servers and leveraging tools like Docker that can help streamline build-release pipelines
